# WorldNetwork
Want to make your own server but can only host 1 server? WorldNetwork is your solution!

#Feature
- Perworld chat
- Global chat
- Perworld fly
- Perworld godmod
- Perworld nickname
- Perworld enderchest
- Perworld kit
- Perworld home
- Perworld Tpa
- Luckperm support
